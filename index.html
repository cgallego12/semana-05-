¿QUÉ HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear y estructurar páginas web. Define la estructura de un documento web mediante etiquetas.

EDITORES DE CÓDIGO

Los editores de código son herramientas que ayudan a los desarrolladores a escribir y organizar su código. Algunos editores comunes son:

  • Visual Studio Code
  • Notepad++
  • Sublime Text

ETIQUETAS HTML5

HTML5 introduce nuevas etiquetas semánticas que ayudan a estructurar el contenido de manera más clara. Algunas de las principales etiquetas son:

  • <header>: Define la cabecera del documento o sección.
  • <nav>: Define una sección de navegación.
  • <article>: Representa contenido independiente y autónomo.
  • <footer>: Define el pie de página del documento o sección.
  • <section>: Define una sección de contenido en el documento.

ETIQUETAS DE CONTENIDO

Las siguientes son algunas de las etiquetas de contenido básicas que se usan en HTML5:

  • <h1> a <h6>: Encabezados de diferentes niveles.
  • <p>: Párrafos de texto.
  • <ul> y <ol>: Listas no ordenadas y ordenadas.
  • <a>: Enlaces a otras páginas o recursos.
